# Phishing Email Detector

An AI-powered phishing email detection tool built with HTML, CSS, and JavaScript, using the Anthropic's Claude API.

## What it does

Analyzes suspicious emails for phishing indicators and returns:
- A verdict: Safe, Suspicious, or Phishing
- A confidence score (0-100%)
- A plain-English summary of findings
- Categorized red flags broken down by Sender, Links, and Content

## Why I built this

Phishing attacks are one of the most common entry points for cybersecurity breaches. This tool demonstrates how large language models can be applied to real-world threat detection, combining my background in forensic case investigation with my studies in Cybersecurity and Applied AI.

## Tech stack

- HTML, CSS, JavaScript (no frameworks)
- Anthropic Claude API (claude-haiku)
- Hosted via GitHub Pages

## How to run it locally

1. Clone this repository
2. Add your Anthropic API key to line 1 of `app.js`
3. Run a local server: `python3 -m http.server 5500`
4. Open `http://localhost:5500` in your browser
